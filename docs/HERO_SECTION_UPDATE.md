## 🔄 **Customization Options**

### **Change the Photo:**
Replace the Unsplash URL in `index.html`:
```html
<img src="YOUR_IMAGE_URL_HERE" 
     alt="Your business description">
```

### **Adjust Gradient:**
Edit in `styles.css`:
```css
.hero-overlay {
  background: linear-gradient(
    135deg, 
    rgba(YOUR_COLOR1) 0%, 
    rgba(YOUR_COLOR2) 100%
  );
}
```

### **Change Icon:**
Replace the SVG in `index.html` with your own icon or logo

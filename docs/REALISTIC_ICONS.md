**Icon Specifications:**
- **Format:** PNG with transparency
- **Size:** 64x64px displayed (can scale)
- **Style:** Photorealistic 3D render
- **Effects:** Drop shadow, glassmorphism
- **Hover:** Scale 1.1x + lift effect

**CSS Enhancements:**
```css

  width: 64px;
  height: 64px;
  filter: drop-shadow(0 4px 12px rgba(102, 126, 234, 0.2));
  transition: all 250ms;
}

.suggestion-card:hover .card-icon-img {
  transform: scale(1.1) translateY(-4px);
```

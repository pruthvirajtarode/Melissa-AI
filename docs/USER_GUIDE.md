## 🚀 Getting Started

### 💻 On Desktop (Windows App)
1. **Download**: Get the `MellissAI Setup 1.0.0.exe` file.
2. **Install**: Double-click the file to install the application.
3. **Launch**: Open MellissAI from your Desktop or Start Menu.

### 📱 On Mobile (Android)
1. **Download**: Get the `app-debug.apk` file.
2. **Install**: Open the file on your Android device (you may need to "Allow from this source").
3. **Launch**: Tap the MellissAI icon on your home screen.

### 🌐 On Web
Simply visit your live Vercel URL (e.g., `https://melissa-ai.vercel.app`).

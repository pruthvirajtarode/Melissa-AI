Professional Chat Widget for Business Development

🚀 Recommended - Clean Embed (Script)

The simplest way to integrate MellissAI. Just add this one line and we handle the rest:

<!-- MellissAI One-Line Integration --> <script src="/js/embed.js"></script>
💡 Tip: Replace /js/embed.js with your full domain (e.g., https://your-bot.com/js/embed.js) if using on an external site.

📋 Advanced - Manual Embed (HTML + Script)

Copy and paste this code if you want full control over the button styling and behavior:

<!-- MellissAI Professional Chat Widget --> <div id="melliss-ai-bot" style="position: fixed; bottom: 25px; right: 25px; z-index: 999999;"> <!-- Floating Button --> <button id="melliss-btn" style=" width: 65px; height: 65px; border-radius: 50%; background: linear-gradient(135deg, #14532d 0%, #22c55e 100%); border: none; cursor: pointer; box-shadow: 0 4px 20px rgba(34, 197, 94, 0.35); display: flex; align-items: center; justify-content: center; transition: all 0.3s cubic-bezier(0.175, 0.885, 0.32, 1.275); " > <!-- Chat Icon --> <svg width="32" height="32" viewBox="0 0 24 24" fill="none" stroke="white" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" > <path d="M21 15a2 2 0 0 1-2 2H7l-4 4V5a2 2 0 0 1 2-2h14a2 2 0 0 1 2 2z"></path> </svg> </button> <!-- Chat Widget (loads in iframe) --> <iframe id="melliss-iframe" src="https://your-domain.com/widget.html" style=" position: fixed; bottom: 100px; right: 25px; width: 400px; height: 600px; border: none; border-radius: 16px; box-shadow: 0 8px 32px rgba(0, 0, 0, 0.15); display: none; z-index: 999998; " allow="microphone; camera" ></iframe> </div> <!-- Widget Control Script --> <script> (function() { const btn = document.getElementById('melliss-btn'); const iframe = document.getElementById('melliss-iframe'); btn.addEventListener('click', function() { if (iframe.style.display === 'none') { iframe.style.display = 'block'; btn.style.opacity = '0'; btn.style.pointerEvents = 'none'; } }); // Listen for close message from iframe window.addEventListener('message', function(event) { if (event.data === 'closeWidget') { iframe.style.display = 'none'; btn.style.opacity = '1'; btn.style.pointerEvents = 'auto'; } }); // Add hover effect btn.addEventListener('mouseenter', function() { this.style.transform = 'scale(1.1)'; }); btn.addEventListener('mouseleave', function() { this.style.transform = 'scale(1)'; }); })(); </script>

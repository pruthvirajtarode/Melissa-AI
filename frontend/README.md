# Frontend

This directory contains all the frontend files for the Melissa AI Business Development Chatbot.

## Structure

```
frontend/
├── index.html          # Main chat interface
├── admin.html          # Admin dashboard
├── widget.html         # Embeddable widget
├── css/                # Stylesheets
│   ├── admin.css      # Admin dashboard styles
│   ├── styles.css     # Main app styles
│   └── widget.css     # Widget styles
├── js/                 # JavaScript files
│   ├── admin.js       # Admin dashboard logic
│   ├── chat.js        # Chat interface logic
│   └── widget.js      # Widget logic
├── images/            # Images and assets
└── manifest.json      # PWA manifest

```

## Features

- **Chat Interface**: Modern, responsive chat UI for interacting with Melissa AI
- **Admin Dashboard**: Management interface for uploaded documents
- **Widget**: Embeddable chatbot widget for external websites
- **Progressive Web App**: Installable web app with offline support

## Technologies

- Vanilla JavaScript (ES6+)
- CSS3 with modern features
- Responsive design for all devices
